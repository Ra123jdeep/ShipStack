# ShipStack - The AI Builder Operating System

ShipStack is a premium AI SaaS platform that helps developers explore and generate tech stacks.

## Project Structure

- `frontend/`: Next.js 14 application with TailwindCSS and Framer Motion.
- `backend/`: FastAPI application with PostgreSQL and AI features.

## Getting Started

### Backend

1. Navigate to `backend/`
2. Install dependencies: `pip install -r requirements.txt`
3. Run server: `uvicorn app.main:app --reload`

### Frontend

1. Navigate to `frontend/`
2. Install dependencies: `npm install`
3. Run server: `npm run dev`

## Features

- AI Stack Generator
- Trust Index System
- Semantic Search
